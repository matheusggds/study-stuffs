### Command Line
- **cd** - change directory
- **ls** - list directory
- **pwd** - print working
- **mkdir** - make directory
- **rmdir** - remove directory

- **cat** - show file contents
- **man** - command manual
- **less** - show file contents by page
- **rm** - remove file
- **echo** - repeat file
